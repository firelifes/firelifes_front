## 模块说明

| 一级模块 | 功能概要 |
|----------|----------|
| **明细/首页** | 本月收支总览、月份切换、按日期分组的收支流水列表 |
| **记账页面** | 收入/支出切换、分类网格选择、金额输入、账户选择、日期选择 |
| **图表分析** | 统计图表展示（趋势图、饼图、排行） |
| **发现页面** | 预算管理入口、FIRE 功能入口 |
| **我的/设置** | 类别设置（大类+分类管理）、账户设置（新增/修改/调整余额）、退出登录 |

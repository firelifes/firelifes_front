## 步骤说明

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 点击记账按钮 (+) | 从首页底部 + 按钮进入记账 |
| 2 | 支出/收入切换 | 顶部 Tab 切换记账类型 |
| 3 | 选择分类 | 从分类网格中选择一级大类，再选择具体二级分类 |
| 4 | 输入金额 | 通过数字键盘输入金额 |
| 5 | 添加备注（可选） | 可为该笔记录添加文字备注 |
| 6 | 选择账户 | 选择资金账户（默认上次使用的账户） |
| 7 | 选择日期 | 默认当天，可自定义日期 |
| 8 | 保存确认 | 点击确认写入数据，返回首页并刷新列表 |

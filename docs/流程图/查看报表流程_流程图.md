## 步骤说明

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 进入图表页 | 从首页导航进入报表分析页面 |
| 2 | 选择时间维度 | 切换本周 / 本月 / 本年，刷新所有图表数据 |
| 3 | 查看支出趋势 | 折线图展示日/周/月支出变化趋势 |
| 4 | 查看分类占比 | 环形饼图展示各分类支出占比 |
| 5 | 查看支出排行 | 条形图展示支出 Top 分类排行 |
| 6 | 筛选特定分类 | 按分类筛选图表数据，聚焦特定分类分析 |
| 7 | 导出报表（可选） | 支持导出 Excel / PDF / 图片格式 |
| 8 | 完成 | 报表查看结束 |

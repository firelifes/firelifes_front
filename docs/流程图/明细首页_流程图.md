## 操作说明

| 操作入口 | 说明 |
|----------|------|
| **切换月份** | 通过月份选择器切换到其他月份，重新加载该月收支数据和流水列表 |
| **点击记账 (+)** | 跳转到记账页面，完成记账后返回并刷新 |
| **点击流水记录** | 进入记录详情，可选择编辑或删除 |
| **编辑记录** | 修改分类、金额、备注、日期、账户，保存后刷新列表 |
| **删除记录** | 弹出确认框，确认后删除并刷新列表；取消则返回记录操作选择 |

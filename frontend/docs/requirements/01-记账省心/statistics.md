# 统计分析页面

## 1. 需求概述

### 1.1 功能定位
统计分析页面是用户查看月度收支统计、消费分类占比和趋势分析的核心页面，帮助用户了解自己的消费习惯和财务状况。

### 1.2 业务目标
- 提供直观的月度收支概览
- 展示消费分类占比（圆环图）
- 展示收支趋势（折线图）
- 展示消费分类排行榜

### 1.3 页面路径
`pages/statistics/index`

## 4. 交互设计

### 4.1 时间切换
- 用户点击时间Tab，切换展示对应月份数据
- 支持近6个月数据切换

### 4.2 类型切换
- 用户选择支出/收入/全部，切换统计维度

### 4.3 排行榜点击
- 点击排行榜项，可跳转至该分类的明细页面

## 5. 接口说明

### 5.1 获取统计数据

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| month | string | 是 | 月份，格式：YYYY-MM |
| type | string | 否 | 类型：expense/income/all |

## 6. 数据流转

```
用户进入页面
    │
    ▼
请求当月统计数据
    │
    ▼
解析数据
    │
    ├──→ 渲染概览卡片
    ├──→ 渲染圆环图 + 图例
    ├──→ 渲染趋势图
    └──→ 渲染排行榜
    │
    ▼
用户切换时间/类型
    │
    ▼
重新请求数据 → 更新视图
```
